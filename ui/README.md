## Install project
1) Install [node.js](https://nodejs.org/en/) 
2) Install dependencies `yarn`

## Run project
`yarn start`
> Project will be run in **http://localhost:8080/**

## Available scripts
```bash
  start     - run dev server
  lint      - run linter
  lint:fix  - run linter with --fix flag
```
